# Pathfinder Reference API

## Setup

This repository uses Vscode Dev Containers to allow for a more seamless developer experience you can find more details on Vscode's [Website](https://code.visualstudio.com/docs/remote/containers) but here is the TLDR.

1. Install [docker](https://docs.docker.com/get-docker/)
2. Install [VSCode](https://code.visualstudio.com/download)
3. Install the [Remote - Containers](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers) vscode extension
4. Clone the repo and open in vscode
5. VScode should prompt you with a notification in the right bottom corner to reopen in a remote container if it does not you can hit Ctrl + Shift + P and enter "Reopen in Container"
6. The container will build and open
